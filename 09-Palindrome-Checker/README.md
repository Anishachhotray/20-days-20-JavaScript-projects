# Palindrome Checker

A simple web application built using **HTML, CSS, and JavaScript** that checks whether a given word, sentence, or number is a palindrome.

A palindrome is a string that reads the same forward and backward, ignoring spaces, punctuation, and case sensitivity.


## Features

 ✔️ Checks words, numbers, and sentences
 ✔️ Ignores spaces and special characters
 ✔️ Case-insensitive comparison
 ✔️ Displays reversed string
 ✔️ Shows clear result (Palindrome / Not Palindrome)
 ✔️ Input validation with user-friendly messages
 ✔️ Simple and responsive UI

## How It Works

1. User enters input text  
2. Input is cleaned:
   - Converted to lowercase  
   - Removes spaces & special characters  
3. String is reversed using JavaScript  
4. Original cleaned string is compared with reversed string  
5. Result is displayed on screen  

## Technologies Used

- HTML5  
- CSS3  
- JavaScript (Vanilla JS)
