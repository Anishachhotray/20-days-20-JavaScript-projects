# Digital Clock

A real-time Digital Clock built using HTML, CSS, and JavaScript.

## Features

* Displays the current time in real-time
* Updates automatically every second
* Two-digit time format (09:05:03)
* Responsive design
* Modern dark-themed user interface

## Technologies Used

* HTML5
* CSS3
* JavaScript

## Concepts Practiced

* DOM Manipulation
* JavaScript Functions
* Date Object
* setInterval()
* String Methods (padStart)
* Dynamic Content Updates
* Real-Time UI Updates

## How It Works

* The application uses JavaScript's Date object to retrieve the current hour, minute, and second.

* The `setInterval()` function updates the clock every second, ensuring that the displayed time always remains accurate.

* The `padStart()` method is used to maintain a professional two-digit format for hours, minutes, and seconds.
