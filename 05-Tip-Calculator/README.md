# Tip Calculator

A simple Tip Calculator built using HTML, CSS, and JavaScript.

## Features

* Calculate tip amount instantly
* Calculate total payable amount
* Input validation for empty fields
* Prevents invalid bill amounts
* Prevents negative tip percentages
* Responsive user interface

## Technologies Used

* HTML
* CSS
* JavaScript

## Concepts Practiced

* DOM Manipulation
* Event Listeners
* Input Validation
* Mathematical Calculations
* Dynamic Content Updates

## How It Works

1. Enter the bill amount.
2. Enter the tip percentage.
3. Click the Calculate button.
4. The application displays:

   * Tip Amount
   * Total Amount
