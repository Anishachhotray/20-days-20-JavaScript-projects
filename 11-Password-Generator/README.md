# Password Generator

A JavaScript-based Password Generator that creates secure random passwords and allows users to copy them instantly.

## Features

* Generate random passwords
* Copy password to clipboard
* Input validation
* Maximum length limit
* Auto-reset on empty input

## Technologies

* HTML
* CSS
* JavaScript

## Learning Outcomes

* DOM Manipulation
* Event Handling
* Clipboard API
* Random Password Generation
* Input Validation
* Using trim() for user input handling
