# Text Analyzer & Counter

A simple Text Analyzer application built using HTML, CSS, and JavaScript.

## Features

* Character Counter
* Word Counter
* Vowel Counter
* Consonant Counter
* Input Validation
* Automatic Reset on Empty Input
* Responsive User Interface

## Technologies Used

* HTML
* CSS
* JavaScript

## Concepts Practiced

* DOM Manipulation
* Event Listeners
* String Methods
* trim()
* split()
* Loops
* Conditional Statements
* Character Analysis

## How It Works

1. Users enter text into the input field.
2. The application counts:

   * Total Characters
   * Total Words
   * Total Vowels
   * Total Consonants

3. The `trim()` method removes unnecessary spaces from the beginning and end of the text.

4. The `split()` method separates text into words for word counting.

5. A loop checks each character to determine whether it is a vowel or consonant.

6. When the input field becomes empty, all counters reset automatically.
