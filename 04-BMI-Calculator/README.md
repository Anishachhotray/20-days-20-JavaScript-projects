# BMI Calculator

- A BMI (Body Mass Index) Calculator built using HTML, CSS, and JavaScript.

# Features

* Calculate BMI using weight and height
* Input validation
* Health status classification
* Responsive user interface
* Dynamic result display

# Technologies Used

- HTML5
- CSS3
- JavaScript

# Concepts Practiced

* DOM Manipulation
* Event Handling
* Conditional Statements
* Input Validation
* Mathematical Calculations
* Dynamic Content Updates

# BMI Categories

- Underweight (BMI < 18.5)
- Healthy (18.5 - 24.9)
- Overweight (25 - 29.9)
- Obese (BMI ≥ 30)
