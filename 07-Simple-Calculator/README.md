# Simple Calculator

A responsive calculator built using HTML, CSS, and JavaScript.

## Features

* Addition, Subtraction, Multiplication, and Division
* Percentage Calculations
* Clear Display Function
* Backspace/Delete Function
* Decimal Number Support
* Error Handling for Invalid Expressions
* Responsive Design

## Technologies Used

* HTML
* CSS
* JavaScript

## Concepts Practiced

* DOM Manipulation
* Functions
* Event Handling
* String Manipulation
* Error Handling
* Arithmetic Operations
* Calculator Logic
* Dynamic Display Updates

## How It Works

1. Users click calculator buttons to build an expression.
2. JavaScript stores the expression dynamically.
3. When "=" is pressed, the expression is evaluated.
4. The result is displayed instantly.
5. Invalid expressions are handled using try-catch and display an error message.
